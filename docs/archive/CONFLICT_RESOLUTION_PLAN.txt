ÉTAPES DE RÉSOLUTION:
─────────────────────────────────────────────────────────────────

ÉTAPE 1: Fixer le tracking upstream de develop
  git branch -u origin/develop develop

ÉTAPE 2: Committer les changements sur develop
  git add .
  git commit -m "feat: v2.2.0 dark mode styling and combobox fixes"
  git push origin develop

ÉTAPE 3: Vérifier la relation entre release/v2.1.0 et main
  git log main..release/v2.1.0
  git diff main release/v2.1.0

ÉTAPE 4: Résoudre les conflits (s'il y en a)
  git checkout release/v2.1.0
  git merge main
  # Résoudre les conflits si nécessaire
  git add .
  git commit -m "merge: resolve conflicts with main"
  git push origin release/v2.1.0

ÉTAPE 5: Mettre à jour main si nécessaire
  git checkout main
  git merge release/v2.1.0
  git push origin main
