# 📦 Archive - Documentation Historique

Ce dossier contient la documentation archivée et les fichiers historiques du projet.

## 📚 Fichiers Archivés

Contient:
- Documentation de versions antérieures
- Fichiers de session (work summaries)
- Notes de release anciennes
- Documentation obsolète
- Brouillons et explorations

## 📋 Organisation

Les fichiers sont archivés pour:
- **Préserver l'historique du projet**
- **Garder la documentation active propre**
- **Permettre la consultation si nécessaire**
- **Faciliter la migration vers les nouvelles versions**

## 🔍 Comment Trouver des Infos

### Si vous cherchez du contenu archivé:
1. Consultez les noms de fichiers avec numéros de version
2. Recherchez dans l'archive si absent de la documentation active
3. Vérifiez les dates de fichier pour le contexte temporel
4. Consultez les fichiers de session pour l'historique des décisions

### Informations Utiles:
- **CHANGELOG.md** - Pour l'historique des versions publiques
- **RELEASES.md** - Pour les notes de release
- Les fichiers de session - Pour comprendre les décisions de design
