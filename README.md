# Center Domiciliation App

Application PHP compatible XAMPP pour la gestion des societes, associes, contrats et collaborateurs.

## Demarrage rapide

1. Copier le projet dans `C:\xampp\htdocs\Center-Domiciliation-App`
2. Demarrer `Apache` et `MySQL` depuis XAMPP
3. Importer `database/schema.sql` dans phpMyAdmin
4. Ajuster `config/database.php` si necessaire
5. Ouvrir `http://localhost/Center-Domiciliation-App/`

Documentation complete: [docs/setup/XAMPP_SETUP.md](docs/setup/XAMPP_SETUP.md)
